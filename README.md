## Instalação

use npm:
```
npm install
```

use yarn:

```
yarn install
```

## Biuld

use npm:
```
npm run tsc
```

use yarn:
```
yarn tsc
```

## Run

use npm:
```
npm start
```

use yarn:
```
yarn start
```
